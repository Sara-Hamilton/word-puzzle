# Word Puzzle

#### takes in string and removes vowels, 1-17-18

#### _By Sara Hamilton and Bryce Frazier_

## Description

_This website takes in a string and returns the string with all of the vowels removed.  It was created to practice for loops in JavaScript_

## Setup/Installation Requirements

* _Clone repository_

* _Open in text editor_

* _Open in browser
